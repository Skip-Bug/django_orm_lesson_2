# Пульт охраны банка
Это внутренний репозиторий для сотрудников банка "Сияние". Если вы попали в
этот репозиторий случайно, то вы не сможете его запустить, т.к. у вас нет доступа к БД,
но можете свободно использовать код верстки или посмотреть как реализованы запросы к БД.

Пульт охраны - это сайт, который можно подключить к удаленной базе данных с визитами и   
карточками пропуска сотрудников нашего банка.   
На сайте можно отслеживать подозрительную деятельность по времени нахождения.   
Просмотреть всех активных пользователей по пропускам.

### Как установить

Часть настроек проекта берётся из переменных окружения.   
Чтобы их определить, создайте файл .env рядом с manage.py   
и запишите туда данные в таком формате: ПЕРЕМЕННАЯ=значение.   
```
DB_ENGINE=база данных
DB_HOST=хост
DB_PORT=порт
DB_NAME=имя базы
DB_USER=имя пользователя
DB_PASSWORD=пароль
SECRET_KEY=код шифроваания
```

Python3 версии >=3.10 должен быть уже установлен.
Проверить версию Python3 можно:
```
python --version 
```
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```

### Как запустить
```
python manage.py startserver
```
Для работы с сервером могут понадобиться и другие команды их можно найти [здесь](ttps://docs.djangoproject.com/en/5.2/ref/django-admin/)

После этого переходите по ссылке [127.0.0.1:8000](http://127.0.0.1:8000), вы увидите главную страницу.
